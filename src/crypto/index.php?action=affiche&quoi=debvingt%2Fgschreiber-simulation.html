La simulation d'une machine G-Schreiber

Ci-dessous, vous trouverez une petite application qui simule le comportement d'une machine G-Schreiber. Le fonctionnement est exactement identique, les cinq premières roues correspondent aux roues qui commandent l'opération d'addition, les cinq roues suivantes aux roues qui commandent l'opération de permutation. La seule différence est que la rotation des roues après chaque caractère n'est pas automatique (pour laisser le temps de comprendre comment la transformation a été effectuée). Il faut cliquer sur le bouton correspondant!
